- virtualenv安装  
虚拟环境的包是对真实环境包的一个复制  
```
sudo pip install virtualenv  # 安装virtualenv
virtualenv venv  # 启动一个虚拟环境
source venv/bin/activate  # 生效一个虚拟环境
(venv)>which python
> /home.../bin/python
(venv)>deactivate  # 退出虚拟环境
```

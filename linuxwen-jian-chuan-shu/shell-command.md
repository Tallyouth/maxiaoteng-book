# 

## 开头

```
#!/bin/sh
```

脚本以.sh结尾, 并且要使用```chmod +x filename.sh```来使文件为可执行脚本


## 注释

注释以" # " 来开始, 占据一行

## 变量


1. 定义变量时,变量名不加美元符号($)
``` 
a="shell" 
```
变量名和等号之间不能有空格, 除了显式赋值, 还可以用语句赋值
```
for file in 'ls /etc'
```

2. 使用变量用${}表示(花括号虽然可以不用, 但尽量使用)
```
echo "a is ${a}"
```

## 环境变量

使用export导入环境变量, 命令可以直接使用

```
# 运行命令的两种方法

/usr/bin/python /XXX/xx.py

export

```

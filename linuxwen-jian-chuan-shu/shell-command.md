# 

## 开头

```
#!/bin/sh
```

脚本以.sh结尾, 并且要使用```chmod +x filename.sh```来使文件为可执行脚本


## 注释

注释以" # " 来开始, 占据一行

## 变量

```
a = "shell"
#输出命令用${}表示
echo "a is ${a}"
```

## 环境变量

使用export导入环境变量, 命令可以直接使用

```
# 运行命令的两种方法

/usr/bin/python /XXX/xx.py

export

```
